OLED-js Font Foundry

Create custom font modules for use with oled-js!

Instructions for use:

  • Type in your chosen font name. Must be a monospaced font.
  • Choose a font size using the slider.
  • Choose a threshold amount. This affects the sampling process when the font is converted to monochrome.
  • Once you are happy with your font, click 'Download font file' and save the output as a .js (JavaScript) file.
  • Require this file as you normally would when using fonts with oled-js.
